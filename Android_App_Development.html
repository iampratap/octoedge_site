As there are three different types of Web Apps

  1. Native App:                                                                                                                                                                   Native app is one that is designed for a particular operating system whether the android or IOS. They are designed in a way that it uses the other latest technology for working like the GPS system etc. It is coded in a specific coding language and uses the code preliminary.                                                                                                                                                                                                                             
  2. Web app:                                                                                                                                                                   Web apps are not real applications; they're really websites that, in many ways, look and feel like native applications, however are not enforced as such. They’re run by a browser and usually written in HTML5. Users initial access them as they might access any web page: they navigate to a special URL and then have the option of “installing” them on their home screen by making a bookmark to that page                                                                                                                                                          
  3. Hybrid App:                                                                                                                                                               They are basically web apps wrapped in a native programming language. The procedure for the development of a hybrid app is the same as the websites. There are a variety of frameworks available now to build hybrid apps.
